Dataset Details

Information about the MRI image dataset used to train and test the model.

🧠 Dataset Summary
  • Total images: ~7000
  • Glioma: ~1,621
  • Meningioma: ~1,646
  • Pituitary: ~1,645
  • No Tumor: ~1,645
  • Format: JPG
  • Type: T1-weighted contrast-enhanced MRI brain scans

⚠️ Limitations

This dataset is intended for demonstration. For clinical applications, much larger datasets with medical validation are required.
