About This Project

An academic deep learning project to detect brain tumors from MRI scans.

🎯 Objective

This project aims to provide a lightweight and effective tool for detecting brain tumors using Convolutional Neural Networks (CNNs). It serves as a demonstration of real-world deep learning applications in medical imaging.

💻 Technologies Used
  • Python, Flask (Web framework)
  • TensorFlow / Keras (Deep Learning)
  • NumPy, OpenCV (Image preprocessing)
  • Bootstrap 5, HTML/CSS (Frontend UI)
🎓 Academic Context

This work is submitted as a final-year project for the TYBSc Computer Science program.
